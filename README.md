# IMCWebApp
Esta app fue el intento a la creacion de una aplicacion de IMC, tiene las clases creadas y quiero creer que funciona pero como no logre vincular la base de datos con el proyecto no logre progresar mas, disculpen la incompetencia seguire practicando para mejorar.

El objetivo
Era crear una applicacion web que permitiera al usuario, descubrir su IMC tomando en cuenta su peso y altura para asi mostrarle un resultado, este inluye un botton para registrar un usuario, una pagina para hacer login y una tabla con historial de los resultados.

Requisitos
Sistema operativo: windows, masOS o Linux
NetBeans IDE Version 15 o superior
Repositorio Git (solo para la clonacion)
